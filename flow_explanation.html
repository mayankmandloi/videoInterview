FastAPI Orchestrator

The central hub managing authentication, database persistence, and external API coordination.

LiveKit Agents

Real-time AI participants that join video rooms as autonomous technical interviewers.

Azure OpenAI

Powers the underlying intelligence for both real-time conversation and post-interview analysis.

Deep Dive: Workflows

Admin Experience

1

Secure Login

Recruiters authenticate using the admin dashboard to manage candidate pools.

2

Interview Configuration

Admins define the role (e.g., "Senior Backend Engineer") and specific skills (e.g., "Python, Distributed Systems") for the AI to assess.

3

Report Review

Structured evaluations are presented with scores, strengths, and hire/no-hire recommendations.

Candidate Journey

1

Unique Access

Candidates join via a tokenized URL that instantly sets up their video and audio peripherals.

2

Live AI Dispatch

Upon starting, the server dispatches a dedicated AI agent to the LiveKit room to conduct the interview.

3

Real-time Assessment

The AI uses voice-to-voice interaction to probe the candidate's technical depth, capturing transcripts for analysis.

Technology Stack

Python 3.10+ FastAPI SQLAlchemy LiveKit Python SDK Azure OpenAI SDK Jinja2

The application is built for high performance and low-latency interaction. By offloading heavy AI processing to specialized providers (LiveKit Inference, Azure), the core application remains lightweight and scalable.
